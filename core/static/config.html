Account Pool Configuration

Network mode: 勾选“使用代理”表示浏览器通过代理出口访问；取消勾选表示直连，直接使用当前机器自身 IP。 直连模式下不要在同一个组里配置大量账号，否则站点很容易识别为“同一 IP 下多个账号”，导致风控甚至封 IP。
When "Use proxy" is checked, the browser accesses the site via the proxy; when unchecked, it uses this machine's own IP. Do NOT put many accounts in one direct-connection group, or the site may treat them as multiple accounts under the same IP and trigger risk control or IP ban.
